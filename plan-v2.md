# 多人弹球游戏 - 新规则

## 核心机制
- 每名玩家有独立的球（初始1个）
- 所有玩家挡板可击打任意球
- 球飞出底部 → 最后击打者得分
- 玩家得分达阈值（每100分）→ 奖励新球
- 玩家无球 → 淘汰（退出游戏）
- 关卡随机生成（rougelike），无尽模式

## 淘汰机制
- 玩家初始有 `balls: [{active: true}]`
- 球出界后从数组移除
- 当 `player.balls.length === 0` → 玩家淘汰，从游戏移除
- 淘汰后挡板消失，不再参与

## 奖励机制
- 玩家得分每达100分，奖励1颗新球
- 新球从该玩家挡板上方发射
- 奖励后球数上限+1，立即生成一个active球

## 关卡生成
- 每次所有砖块消除后，随机生成新关卡
- 随机性：砖块数量、排列、颜色分布
- 难度递增：随着关卡数，砖块数量可能增加

## 数据模型
```js
Player {
    id, color, x, y, score, balls: [{x, y, dx, dy, lastHitter}],
    ballsRemaining: 1, maxBalls: 1+floor(score/100)
}

Ball {
    x, y, dx, dy, lastHitter: playerId
}
```

## 需要实现
- 服务器 authoritative 物理引擎
- 每玩家独立球列表
- 最后击打者追踪
- 随机砖块生成算法
- 关卡进度系统
- 无尽模式难度曲线
